# Pirate-Intelligent-Agent

## What is this project

This is a project I created using neural networks to create a reinforcement learning algorithm in order to create an AI agent that can with 100% accurracy complete a maze path game.
I was given the initialy functionality and basic functions to allow me to create an algorithm to achieve the projects overall goal. I created the algorithm, neural networks, and set the game loop 
so the agent could create new actions, and learn on previous actions to achieve its goals.
